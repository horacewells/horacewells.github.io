Activity: 12 August 2024 - XSS vulnerability exposed by the DeepL extension

Title: XSS vulnerability exposed by the DeepL extension
Date: 2024-08-12
Tags: Vulnerability

A Cross-Site Scripting (XSS) vulnerability has been identified in the DeepL browser extension. This vulnerability primarily affects the Google Chrome and Edge version of the extension, but has not been observed in other browsers. The issue allows the execution of HTML tags embedded in arbitrary text. To exploit this vulnerability, it is sufficient to highlight the relevant text using the mouse cursor.
